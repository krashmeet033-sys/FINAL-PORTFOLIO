# 🌸 Rashmeet Kaur | Portfolio

A modern, responsive, and interactive personal portfolio website built using React.js and Vite. This portfolio showcases my technical skills, projects, achievements, learning journey, and passion for creating elegant digital experiences through code and creativity.

The website is designed to provide a professional online presence while highlighting my growth as a Computer Science student, frontend developer, and problem solver.

# 📖 About The Project

This portfolio was developed to create a professional digital identity and showcase my technical journey as a B.Tech Computer Science student.

The website highlights my projects, skills, achievements, and learning experiences while demonstrating modern frontend development practices. The goal was to build a portfolio that is visually appealing, responsive, and user-friendly while maintaining clean code architecture and performance.

The portfolio includes:

* Home Section
* About Me
* Personal Strengths
* Technical Toolkit
* Projects Showcase
* Currently Learning
* Achievements
* Contact Information

# ⚙️ Installation & Setup

### Clone Repository

git clone YOUR_REPOSITORY_LINK

### Navigate to Project Folder

cd portfolio

### Install Dependencies

npm install

### Run Development Server

npm run dev

### Build Project

npm run build

# 🌍 Deployment

The portfolio is deployed using Vercel.

Deployment Workflow :- 

1. Source code pushed to GitHub
2. GitHub repository connected to Vercel
3. Automatic build and deployment
4. Live website generated online
